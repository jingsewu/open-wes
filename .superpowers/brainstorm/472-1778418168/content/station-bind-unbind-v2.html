完整流程（修订版）

首次进入（未绑定）：进入工作站路由 → isStationSelected=false → 显示绑定页 → 选择 → 确认 → 进入作业
再次进入（已绑定）：进入工作站路由 → isStationSelected=true → 直接进入作业（跳过绑定页）
退出：Header "退出工作站" → 确认 → OFFLINE → 自动跳回卡片页 → 出现解绑 banner
解绑：Banner "解绑工作站" → clearStation() → banner 消失 → 下次进入显示绑定页
保留绑定：Banner "保留绑定" / 忽略 → 下次点同一台直接进入作业
